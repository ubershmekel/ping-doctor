# Privacy Policy

Last updated: April 1, 2026

## Overview

PingDoctor is a browser extension that helps users monitor the availability and
response time of user-configured HTTP and HTTPS endpoints, including local
network devices such as a router.

PingDoctor is designed to keep its data local to the user's browser.

## Data PingDoctor Uses

PingDoctor stores the following information locally in the browser:

- User-configured monitoring targets, labels, and polling interval
- Recent connectivity samples and response-time history
- Derived outage history and daily summaries
- Local state needed to keep monitoring working across sessions

This information is used only to provide the extension's monitoring features.

## Data Collection and Sharing

PingDoctor does not collect personal information for the developer.

PingDoctor does not sell user data.

PingDoctor does not transfer user data to third parties, except for the network
requests the extension makes directly to the monitoring targets configured by
the user.

## Network Requests

PingDoctor sends HTTP or HTTPS requests only to endpoints the user configures
for monitoring. These requests are used to determine whether a target is
reachable and how long it takes to respond.

Examples may include:

- A local router or gateway address such as `http://192.168.1.1`
- A public internet health-check URL chosen by the user

PingDoctor does not use those requests to download or execute remote code.

## Storage

PingDoctor uses browser storage APIs to persist settings and monitoring history
so the user can view status and trends over time.

Stored data remains on the user's device unless the user exports it or browser
sync or backup features outside the extension copy that data.

## Remote Code

PingDoctor does not use remote code. All executable extension code is packaged
with the published extension.

## User Controls

Users can control PingDoctor's data by:

- Changing or removing monitored targets
- Clearing stored monitoring history
- Uninstalling the extension

## Changes to This Policy

If this policy changes, the updated version will be published in this document
with a new "Last updated" date.

## Contact

For questions about this policy, contact the extension publisher through the
